2. DEFINITIONS
“Personal Data”
Any information relating to an identified or identifiable natural person pursuant to GDPR Article 4 and CCPA/CPRA definitions.
“Processing”
Any operation performed on Personal Data, including access, storage, transmission, deletion, or modification.
“Controller”
The Client — the entity that determines the purpose and means of personal data processing.
“Processor / Service Provider”
DEVIN BENARD ROYAL, limited to support/services operations.
“Subprocessor”
Any third party authorized to process Personal Data on behalf of the Processor.
Default stance: Processor does NOT use subprocessors unless explicitly authorized in writing.
“Applicable Privacy Laws”
Includes but is not limited to:
GDPR (EU)
UK-GDPR
CCPA / CPRA
VCDPA
CPA
HIPAA Security-Adjacent Controls (where applicable)
Global enterprise privacy frameworks
3. SCOPE OF DATA PROCESSING
3.1 Nature of Processing
Processing is limited to:
Technical logs
Diagnostic metadata
Operational troubleshooting
Support interactions
Enterprise-OS™ itself performs no autonomous data collection, no telemetry, no analytics, and maintains zero outbound network transmissions.
3.2 Purpose of Processing
Processor may process Personal Data solely for:
Providing support & maintenance
Troubleshooting software issues
Executing SOW-defined services
Ensuring contractually required functionality
3.3 Duration
Processing continues only for:
The Support Term
The Service Term
The License Term
After expiration, all data must be deleted or returned unless prohibited by law.
4. DATA MINIMIZATION & PRIVACY BY DESIGN
Enterprise-OS™ is engineered to:
✔ Perform NO data tracking
✔ Perform NO unauthorized logging
✔ Contain NO telemetry
✔ Make NO external network calls
✔ Maintain ZERO external dependencies
All processing is:
intentional
limited
controller-directed
strictly necessary for service delivery

5.1 Follow Controller’s instructions only

5.2 Maintain confidentiality

5.3 Implement Technical & Organizational Measures (TOMs)
Including:

5.4 Assist Controller with compliance
